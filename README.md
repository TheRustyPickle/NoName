# What Is This?
<!-- rumdl-disable-file MD013 -->

A dumpster fire project to experiment with Web3 alongside an attempt at gaining experience with Redis. The project contains a frontend created with Next.js and a websocket backend created with Rust + Actix-Web. A good chunk of the frontend was created with AI.

## What It Does

* Contains 4 games Tetris, 2048, Snake and Flappy Bird for gaining points with increasing difficulty
* A real-time leaderboard that updates on any point changes for the top players
* A referral program that allows gaining % of points from referrals
* Backend that verifies game moves from the frontend and ensures it's valid
* A small task system for gaining points
* Uses ETH or Solana wallet to log in
